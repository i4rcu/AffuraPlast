اختر التصميم — ٣ أشكال مختلفة

نفس المحتوى (خبرة قطاعية)، لكن بتصميم وحركة مختلفة لكلٍّ منها. افتحها على هاتفك ثم قل لي: A أو B أو C

A · زمرّدي دافئ
خلفية كريمية + لمسة زمرّدية، بطاقات نظيفة (الموقع الحالي)
افتح ←
B · ستوديو داكن
خلفية داكنة، قائمة قطاعات تفاعلية، لوحات كبيرة، ذهبي/زمرّدي
افتح ←
C · إديتوريال
أبيض، حروف ضخمة، فهرس وخطوط رفيعة، بسيط وأنيق
افتح ←

بعد اختيارك سيصبح هذا هو الموقع النهائي (مع التركية والإنجليزية الكاملة).
